Tags

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

  1. All HTML tags must enclosed within <> these brackets.
  2. Every tag in HTML perform different tasks.
  3. If you have used an open tag , then you must use a close tag (except some tags)

TAG DESCRIPTION
Document summary
html This tag specifies that the webpage is written in HTML. It appears at the very first and last line of the webpage. It is mainly used to show that the page uses HTML5 – the latest version of the language. Also known as the root element, this tag can be thought of as a parent tag for every other tag used in the page.
head This tag is used to specify meta data about the webpage. It includes the webpage’s name, its dependencies (JS and CSS scripts), font usage etc
title As the name suggests, this tag contains the title/name of the webpage. You can see this in your browser’s title bar for every webpage open in the browser. Search engines use this tag to extract the topic of the webpage, which is quite convenient when ranking relevant search results
body Everything the user sees on a webpage is written inside this tag. It is a container for all the contents of the webpage.
Document information
meta This is the meta data tag for the webpage. Can be useful for mentioning the page’s author, keywords, original published date etc.(Singular tag)
Document structure
h1 to h6 Six different variations of writing a heading. h1 has the largest font size, while h6 has the smallest.
p Plain text is placed inside this tag.Paragraph tag.
br A line break for webpages. Is used when wanting to write a new line.(Singular tag)
hr Similar to the above tag. But in addition to switching to the next line, this tag also draws a horizontal bar to indicate the end of the section.(Singular tag)
Text formatting
b Makes text bold. Used to emphasize a point
i This displays text in italics.
sub Used for writing a subscript (smaller font just below the mid-point of normal font).
sup Similar to the above tag, but for superscripting
a href="url" Anchor tag. Primarily used for including hyperlinks.
a href=”mailto:” Tag dedicated to sending emails.
a name=”name This tag can be used to quickly navigate to a different part of the webpage.
Image
img A tag to display images in the webpage.(Singular tag)
Attributes for img tag and their description
src=”url” The URL or path where the image is located on your drive or on the web.
alt="text" The text written here is displayed when user hovers mouse over the image. Can be used to give additional details of the image.
height="xy" Specifies image height in pixels or percentages.
width="xy" Specifies image width in pixels or percentages.
border=”” Specifies border thickness of the image. If not mentioned, defaults to 0.
Image Mapping
map Denotes an interactive (clickable) image.
map name="" Name of the map associated between the image and the map.
area Specifies image map area.(Singular tag)
shape="" Shape of the area.(Attribute for area tag)
coords="" Coordinates of the vital information of the shape. Example: vertices for rectangles, center/radius for circles.(Attribute for area tag)
Lists
ol Tag for ordered or numbered list of items.
ul Contrary to the above tag, used for unordered list of items.
li Individual item as part of a list.
Table
table Marks a table in a webpage.
thead Specifies information pertaining to specific columns of the table.
tfoot Determines the footer of the table.
tr Denotes a single row in a table.
td A single cell of a table. Contains the actual value/data.
th The value of a heading of a table’s column.
Frames
iframe An inline block of content, this is used as a container for multimedia in a flexible manner. It floats inside a webpage, meaning it is placed relative to other webpage items.
iframe attributes
name="" The name of the iFrame.
src="" The source URL/path of the multimedia object to be held inside the iFrame.
height="" Determines the height of the iFrame.
width="" Determines the width of the iFrame.
